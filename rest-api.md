## REST Api

### Resources:
* Products
* Orders
* Customer


### Endpoints:
* load all products   
GET /api/products
* load product with id  
GET /api/products/{id}
* load all products with tag  
GET /api/products?tag={tag}
* delete a product  
DELETE /api/products/{id}
* create a prodcut  
POST /api/products
* load all customers  
GET /api/customers
* create a customer  
POST /api/customers
